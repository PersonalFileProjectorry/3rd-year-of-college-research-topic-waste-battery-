**기능** 

INA219로 전류/전압 실시간 측정
설정한 임계치 초과 시 서보 모터 작동
온도 센서(TMP36, DS18B20 등)와 연결 가능 (온도 조건 추가 가능)

**4. 코드 설명**
INA219 I2C 초기화 → 라즈베리파이 SDA/SCL 사용
서보 모터 PWM 초기화 → GPIO18 (50Hz)
임계치 판정
전류가 설정치 초과 → 팽창 감지
SWELLING_SIGNAL: 가상 신호, 온도 센서 기준으로 생성 가능
서보 모터 제어
팽창 발생 → 90도 회전
정상 → 0도
실시간 출력
Current, Voltage, Power 출력 → 로그 기록 가능

**5. 확장**
DS18B20 온도 센서 추가 → 70–80°C 구간에서 팽창 신호 생성
가상 전류 모델 적용 → Python 시뮬레이션 로직과 연동
재사용 가능/불가 판단 → CSV 로그 파일에 기록 후 분석

이 코드와 회로를 그대로 사용하면:

1. **실시간 전류/전압 측정**
2. **가상 팽창 신호 기반 서보 제어**
3. **Python으로 로그 + 재사용 판정**

모두 구현 가능합니다.
