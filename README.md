# 공모전용 폐배터리 재사용 판단 실험 설계

## 1. 목표 설정
- **메인 목표:** 재사용 가능한 폐배터리 vs 재사용 불가능한 폐배터리 판별
- **근거:**  
  - 대학 실험 환경에서 열폭주(120~150°C 이상) 테스트는 안전상 불가
  - 초기 열화 신호(팽창, 내부 저항 증가, 전류 변화)를 감지하는 50~80°C 범위에서 실험 가능
- **비고:** 열폭주 위험 판별은 가상 시뮬레이션으로만 진행 가능

---

## 2. 실험 온도 범위 설정
| 구분 | 온도 범위 | 목적 / 의미 |
|------|-----------|------------|
| 초기 열화 감지 | 50–70°C | SEI 분해 및 내부 저항 증가 관찰, 가상 전류/팽창 신호 기준 |
| 재사용 판단 | 70–80°C | 안전 배터리는 손상 없이 유지, 열화 배터리는 팽창/전류 이상 발생 → 재사용 불가 판정 |
| 위험 폭주 구간 | >120°C | 실험 환경에서 불가, 가상 시뮬레이션 필요 |

---

## 3. 실험 구성 (하드웨어)
- **히터:** 12V 50W, STM32F103 PWM 제어, MOSFET AOD4184A + 비아스티칭 10–15개
- **챔버:** 40×30cm → 실험 영역 20×15cm 또는 13×10cm 활용
- **단열재:** 고밀도 폴리우레탄 / 글라스울 → 열 손실 최소화
- **센서:** INA219 (전류/전압), I²C 통신 → 라즈베리파이에서 데이터 수집
- **판정 장치:** 서보 모터 → 가상 팽창 신호 발생 시 동작

---

## 4. 실험 논리
1. **가상 전류/팽창 데이터 생성** (Python 시뮬레이션)
   - 열화 배터리: 10분 내 팽창/전류 이상 → 재사용 불가
   - 정상 배터리: 10분 유지 → 재사용 가능
2. **온도 유지:** 70–80°C 구간에서 10분 관찰
3. **결과 판정:**  
   - 팽창/전류 이상 → 재사용 불가  
   - 이상 없음 → 재사용 가능
4. 배터리- 3
   -초기 열화 감지때 1개 불량발생
   -재사용 판단시점에서 +1 더 발생
   -정상배터리 - 3중 하나

---

## 5. 근거
- **SEI 분해/초기 열화 시작 온도:** 50–60°C (Patel 2020, Chen 2026 리뷰 논문)
- **재사용 가능 배터리 안전 온도:** 70–80°C까지 문제 없음
- **열폭주 임계 온도:** 120–150°C → 대학 실험 환경에서 실험 불가

---

## 6. 추천 실험 범위
- **온도:** 50–80°C  
- **시간:** 10분 유지 관찰  
- **판정 기준:** 가상 전류/팽창 신호 발생 여부
- **장치:** 라즈베리파이 + INA219 + 서보 모터, STM32F103 히터 제어
