4. 설명
시간(time): 0~10분, dt=0.1분 간격으로 시뮬레이션
가상 전류(current):
정상 배터리: ±5% 변동
열화 배터리: 5분 이후 급상승
팽창 신호(swelling):
정상 배터리: 0
열화 배터리: 5분 이후 1 → 서보 모터 동작 시그널
판정 로직:
전류가 임계값(3 A) 초과 또는 팽창 신호 발생 → 재사용 불가
둘 다 정상 범위 → 재사용 가능
5. 활용
라즈베리파이에서 INA219/I²C 센서와 연동하여 실시간 가상 전류 데이터를 모의
서보 모터와 연동하여 팽창 감지 시 동작
50–80°C 범위에서 재사용 가능/불가 판정 시뮬레이션
